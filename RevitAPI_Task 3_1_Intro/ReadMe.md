\# DeleteObject - Плагин для удаления элементов в модели Autodesk Revit



\## Описание плагина



Плагин \*\*DeleteObject\*\* предназначен для удаления выбранных элементов в проекте Autodesk Revit.

\## Основные функции



\- \*\*Удаление выбранных элементов\*\* - удаляет все предварительно выделенные элементы в активном документе

\- \*\*Проверка наличия выделения\*\* - проверяет, выбрал ли пользователь элементы перед выполнением операции

\- \*\*Безопасное выполнение\*\* - производит обработку возможных исключений

\- \*\*Обратная связь с пользователем\*\* - информирует пользователя о ходе выполнения операции и возможных ошибках
